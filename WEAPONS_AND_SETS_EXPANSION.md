## üéÆ Usage in Game

### Weapons Configuration
All weapons are defined in: `Assets/Resources/Config/weapons_expanded.json`

**Structure:**
```json
{
  "id": "weapon_ragnarok",
  "name": "Ragnarok",
  "slot": "weapon",
  "rarity": 5,
  "weaponType": "sword",
  "mainStat": {"atk": 180, "crit": 25, "critDmg": 50, "spd": 15},
  "element": "fire",
  "passive": {
    "name": "World Ender",
    "effect": "25% chance to deal 500% damage to all enemies"
  }
}
```

### Gear Sets Configuration
All sets are defined in: `Assets/Resources/Config/gear_sets_expanded.json`

### Integration
These weapons and sets integrate with:
- **EnhancedGearManager** - For gear generation and set bonuses
- **Loot tables** - For dungeon drops
- **Crafting system** - For recipe creation
- **Gacha/Summon system** - For rare weapon acquisition

## üìà Progression Curve

### Early Game (Level 1-20)
- Common and Uncommon weapons
- No sets available yet
- Focus on basic stats

### Mid Game (Level 21-40)
- Rare and Epic weapons
- First set pieces start dropping
- Passive abilities become important
- 2-piece bonuses

### Late Game (Level 41-60)
- Epic and Legendary weapons
- Complete 4-piece sets
- Hunt for passive combinations

### End Game (Level 61+)
- Legendary, Unique, Mythic weapons
- 6-piece set completions
- Min-maxing weapon passives with set bonuses
- Hunting for perfect rolls
